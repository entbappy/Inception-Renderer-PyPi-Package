# InceptionRender

This package is meant to render Youtube tutorial videos and reference site into the Colab or Jupyter notebooks for easy reference in future.

- PYPI link for this package - [InceptionRender](https://pypi.org/project/InceptionRender/)

## Getting Started

### Installation

!!! note "installation steps"
    First let's do an easy pip installation of the library by running the following command -
    ```bash
    pip install InceptionRender
    ```
